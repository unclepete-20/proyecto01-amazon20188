# Proyecto Clon de Amazon

Este es el Proyecto #1 del curso de Sistemas y Tecnologíias Web del año 2022, impartido en la Universidad del Valle de Guatemala. Dicho proyecto consistió en aplicar los conocimientos y herramientas aprendidas a lo largo del curso para poder recrear una página web de nuestro gusto. En este caso yo, Pedro Pablo Arriola Jiménez con número de carnet 20188, decidí aplicar mis conocimientos y ciertas tecnologías web para recrear una de las páginas de E-Commerce más importantes a nivel global, [Amazon](https://www.amazon.com/-/es/ref=nav_logo#nav-top).
